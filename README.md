# AI-Painting-with-Python
Virtual painter project by using hand landmarks detector. Painter with finger as the brush and our video as the canvas.
Hal yang perlu diperhatikan:
1. Gunakan Python versi <= 10 yaitu 3.8, 3.9, atau 3.10 jika di atas itu akan gagal build
2. Gunakan mediapipe versi 0.10.9
3. Pastikan Numpy versi 1.2x. Karena mediapipe tidak stabil di numpy >= 2.x
